SELFTRUM HC is a TRIBUTE TO ZX Spectrum & HC-85 old consoles.
A Photo Booth based on the esthetics of 8 bit style image processing.
Open Selftrum.pds and run it.
You will fins guidance to take the photo inside the app.
Final photo will be saved in OUTPUT folder.
Enable API to further enhance the 3 photo styles : Normal, BW, 8 BIT.
Config & enable network to print the final photo.
Email is not implemented yet and there is no option in the photo booth !!!
Enjoy !!!
