SELFTRUM HC is a photo booth with the esthetics of 8 bit style image processing, made as a tribute to ZX Spectrum & HC-85 old consoles.

Open Selftrum.pde and run it.

You will find step by step guidance inside the app.

Final photo will be saved in OUTPUT folder.

Add and enable you own GROQ API Key to further enhance the 3 photo styles : Normal, B&W, 8 BIT.

Printing & e-mail are not implemented yet and there is no option in the app for this version !!!

Enjoy !!!
